# Support

`knowledge-agent-mvp` is an experimental phase 0 MVP.

## Getting Help

Use GitHub Issues for:

- Bug reports.
- Reproduction cases.
- Documentation gaps.
- Small feature requests aligned with the phase 0 Agent Runner roadmap.

Use GitHub Discussions if enabled for:

- Questions.
- Design discussion.
- Roadmap suggestions.

## Before Asking

Please check:

- [README.md](README.md)
- [backend/README.md](backend/README.md)
- [frontend/README.md](frontend/README.md)
- [docs/modules/README.md](docs/modules/README.md)

## Not Supported

The current MVP does not provide production support for:

- Hosted deployment.
- Enterprise authentication.
- Multi-tenant permissions.
- Production sandbox isolation.
- Compliance certification.
- Managed LLM credentials.
